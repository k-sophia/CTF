## Event Details

Hosted by Correlation One

Date: June 14th 2025, 11:00AM - 7:00PM ET

Challenges Difficulty:
* Very Easy - 75 points
* Easy - 150 points
* Medium - 200 points
* Hard - 300 points

Tools Used:
* Burp Suite
* [CyberChef](https://gchq.github.io/CyberChef/)
* [dcode.fr/en](http://dcode.fr)
* [DTMF Decoder](https://dtmf.netlify.app/)
* `exiftool`
* `strings`
* TOR Browser
* [jwt.io](http://jwt.io)
* `unzip`
* wireshark
